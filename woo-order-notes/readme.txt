== Description ==

**WPC Order Notes** help you manage all order notes easier. You can see all notes in one place and quick view notes of an order in the popup.

= Features =

- List all notes with paging
- Quick view notes of an order
- WPML integration

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Order Notes", Install & Activate it
4. Now you can open the WooCommerce/Notes page to see all notes
